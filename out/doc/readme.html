HR text tools

Filter programi.
sort_hr
perl skripta za hrvatsko sortiranje
hr_cp1250-to-tex
filter za prebacivanje iz windows-1250 kodne stranice u TeX ascii.
hr_latin2-to-tex
filter za prebacivanje iz latin2 (iso8859-2) kodne stranice u TeX ascii.
hr_tex-to-cp1250
filter za prebacivanje iz TeX ascii u latin2 (iso8859-2) kodnu stranicu.
Za ostale kombinacije prebacivanja može se koristiti GNU recode. Npr. cat testfile.txt | recode cp1250..utf8 > testfile1.txt
