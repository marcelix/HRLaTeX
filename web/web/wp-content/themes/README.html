Tiny - a WordPress theme for minimalists

ReadMe for Tiny theme

This is a minimalist, single column, fixed-width theme for WordPress. You should unpack this in your wp-content/themes folder and select it in the admin panel.

Changing the header image

You get the Tiny theme header image with this package. It is recommended that you make your own header image, sized 330x140px and name it tinyheader.jpg and place it in the theme folder.

Plugins

This theme has code in place to use the ComPreVal (Comment Preview and Validation) plugin. I have also placed the code for the optional theme switcher in the sitemap. You only need to download, install and activate the plugin you want to use.

If you think another plugin could be a good addition to this theme (keeping in mind this is a minimalist theme), please let me know and I will consider adding the necessary code.

Other customizations

I try to comment the stylesheet in a way that shows what is going on. It should be quite straightforward to make your changes to the stylesheet. Also the code for the theme itself is not very complicated. The only oddity is that there is no sidebar, and most of the functionality usually seen in the sidebar is now placed in the sitemap.

If you need help with adjustments to the design, you can hire me to do so, or even to make you a separate design or theme to your specifications. I am a professional freelance web designer/developer who has bills to pay…

Further development

As the theme continues to be in development, certain details may change. I recommend you stay in tune with what's happening on my WordPress development site. If you find problems or shortcomings in this theme, please do not hesitate to tell me, so I can try to fix things.
